## Fifa 2018 Clubs Pro API ##

Fifa 2018 Clubs Pro API, built on NodeJS/ES6.  Wrapper to get stats for Fifa 18 Pro Clubs Teams.


## Installation

```
npm install fifa-api-proclubs
```
